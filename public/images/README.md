# Images Directory

Add your service images here. Recommended image names:

- `tax-services.jpg` - For Tax & Financial Services
- `business-registration.jpg` - For Business Registration
- `certification.jpg` - For Certification & Documentation
- `architectural.jpg` - For Architectural Services
- `home-loan.jpg` - For Home Loan Services

**Image Requirements:**
- Format: JPG, PNG, or WebP
- Recommended size: 800x600 pixels or larger
- Aspect ratio: 4:3 or 16:9
- File size: Under 500KB for optimal performance

If images are not provided, the website will display beautiful gradient backgrounds with icons.
